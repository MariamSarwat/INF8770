IMPORTANT:
data folder must contain the images, videos in these subfolders:

    jpeg
    png
    video

Index files will be generated in these subfolders:

    index
    index_new

To generate histograms corresponding to videos run :
    python3 generate_video_hists.py (question 2)
    python3 generate.py (question 3)

To find images found in jpeg and png folder run (an excel will be generated with the results):
    python3 find_image.py (question 2)
    python3 find.py (question 3)

You will probably need to install :
    pip3 install xlsxwriter
    pip3 install scipy
